# Design Notes

## English

`hello-world.exe` is designed like a small artifact from an alternate computing timeline.

The page borrows from three visual references:

- Early home computers and beige hardware
- CRT monitors with phosphor glow and scanlines
- Fictional hacker consoles from retro science-fiction cinema

The interface keeps the first screen focused on the actual experience. There is no landing-page preface and no marketing-style hero. The terminal is the product, the toy, and the documentation sample all at once.

## 中文

`hello-world.exe` 的设计目标，是让它像一件来自平行计算机历史的小展品。

视觉上主要借了三类气质：

- 早期家用电脑和米色硬件
- CRT 显示器的荧光、扫描线和暗场
- 复古科幻电影里的黑客控制台

页面第一屏直接呈现体验本身，没有营销式介绍，也没有多余的落地页包装。这个终端既是产品，也是玩具，也是说明书里的示例。

# Release Notes

## v1.0.0 - First Stable Boot

`hello-world.exe` has completed its first stable boot.

This release turns the classic "Hello, World!" into a small retro terminal museum: five modes, one glowing CRT screen, and a tiny bit of programming-history theater.

## Highlights

- A polished CRT-inspired interface
- Five Hello World modes: BASIC, C, Python, Enterprise, and Esolang
- Static deployment on GitHub Pages
- No framework, no build step, no dependencies
- Bilingual documentation and project screenshots

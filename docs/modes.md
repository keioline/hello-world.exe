# Mode Guide

## English

`hello-world.exe` currently includes five modes.

## BASIC

The BASIC mode is the most old-school one. It references the ritual of typing a tiny program into a home computer and watching the screen answer back.

## C

The C mode is compact, formal, and canonical. It keeps the braces because sometimes ceremony is part of the charm.

## Python

The Python mode is intentionally plain. One line, one sentence, no fuss.

## Enterprise

The Enterprise mode treats a greeting as if it were production infrastructure. It is a small joke about how modern software can turn a sentence into a platform.

## Esolang

The Esolang mode invents a tiny fictional language. It exists because useless little languages are one of programming's better jokes.

---

## 中文

`hello-world.exe` 目前包含五种模式。

## BASIC

BASIC 模式是最 old school 的版本。它致敬那种把几行程序敲进家用电脑，然后等屏幕回应你的仪式感。

## C

C 模式短小、正式、经典。花括号被保留下来，因为有时候仪式本身就是魅力的一部分。

## Python

Python 模式刻意保持朴素。一行代码，一句话，不多解释。

## Enterprise

Enterprise 模式把一句问候当成生产级基础设施来处理。这是一个关于现代软件如何把一句话包装成平台的小玩笑。

## Esolang

Esolang 模式虚构了一门三命令语言。它存在的理由很简单：没什么用的小语言，是程序世界里很好的一类玩笑。

# hello-world.exe

A retro terminal museum for the oldest small program in the room.

Open `index.html` in a browser and switch between BASIC, C, Python, Enterprise, and Esolang modes.

For a local server:

```bash
python -m http.server 4173
```

Then open `http://127.0.0.1:4173/`.

## Why

Because a Hello World repository does not have to be boring. It can be a little old school, a little theatrical, and still small enough to understand in one sitting.
